# A to-do list with typescript

## Features
- Display tasks
- Create new tasks
- Delete task
- Update task
- Filter tasks by date
- Filter tasks by priority
- Search task
- Create new categories
- Filter tasks by category

## How to
1. Clone this repository.
2. Install npm 
3. Install all the dependencies running pnpm install.

## Compile
```npm start```
